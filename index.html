01 // identidade

Quem somos?

Somos um grupo de alunos movidos por uma mesma vontade: transformar ideias em código. No Developer's Club, exploramos a programação como ferramenta para resolver problemas reais — da sala de aula até ao espaço. Aqui, ninguém precisa de saber tudo. Precisa apenas de querer aprender.

02 // percurso

O que já construímos

Quatro projetos. Quatro provas do que conseguimos fazer juntos.

01 CANSAT

Equipa InAxon

Um satélite do tamanho de uma lata. Um desafio à altura de uma equipa.

Participámos no CanSat, um desafio internacional onde equipas de estudantes constroem um mini-satélite capaz de recolher e transmitir dados durante uma queda controlada. Juntámos eletrónica, software e engenharia para colocar o nosso projeto no ar — e trazer os dados de volta ao chão.

  • hardware
  • sensores
  • software embebido
  • equipa
02 OLIMPÍADA

Náboj Matemática

Mentes rápidas. Raciocínio afinado. Pressão a sério.

Representámos a escola no Náboj, uma competição internacional de matemática em que equipas resolvem problemas exigentes contrarrelógio. Mais do que medir conhecimento, o Náboj testa estratégia, colaboração e a capacidade de pensar com clareza quando o tempo aperta — exatamente o tipo de desafio que nos move.

  • resolução de problemas
  • equipa
  • pensamento rápido
03 PARTILHA

Matemática + Código

Quando o código dá vida às fórmulas.

Levámos à turma uma apresentação sobre sucessões — não em fórmulas estáticas no quadro, mas em código a correr. Mostrámos como a programação pode tornar a matemática mais visual, mais intuitiva e, sobretudo, mais nossa: traduzimos conceitos em pequenos programas e demos vida aos termos das sucessões, um a um.

  • sucessões
  • visualização
  • didática
04 COLABORAÇÃO

Por trás do Futuro

Um website para dar voz a quem está a construir o seu caminho.

Apoiámos o projeto "Por trás do Futuro" no desenvolvimento de um website que reúne entrevistas a estudantes universitários e profissionais — partilhas autênticas sobre escolhas, percursos e o que está por trás de cada futuro. Combinámos design, código e inteligência artificial para entregar uma plataforma funcional, leve e pensada para o leitor.

  • web dev
  • IA como ferramenta
  • design
  • projeto real
